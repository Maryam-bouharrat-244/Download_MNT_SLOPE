## 🆘 Si quelque chose se passe mal

### Message: "GDAL non trouvé"
```
✅ Normal! GDAL optionnel pour conversion GeoTIFF
VRT fonctionne sans GDAL dans QGIS
```

### Message: "Erreur téléchargement"
```
❌ Connexion Internet interrompue
Relancez le script - détecte fichier existant et reprend
```

### Message: "Aucun fichier ASC"
```
❌ Extraction échouée
Supprimez dossier et retéléchargez
Vérifiez espace disque disponible
```

### Mais je vois quand même -99 au lieu de -99999?
```
❌ Ancienne version utilisée
✅ Utilisez create_mnt_vrt_FIXE.py (ou relancez app)
```
