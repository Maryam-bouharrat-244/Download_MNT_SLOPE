# 🎉 Suite Complète de Fusion MNT - RÉSUMÉ

## ✅ Qu'est-ce qui a été créé?

Un ensemble complet de **scripts Python et outils** pour **fusionner 291 tuiles raster du MNT FRANCE** en un seul fichier.

### 📂 Fichiers créés dans `C:\Users\marya\Downloads\`

#### 1️⃣ **Scripts Principaux** (à exécuter)

- **`create_mnt_vrt.py`** ⭐ 
  - **Rôle:** Crée une mosaïque VRT à partir des 291 tuiles
  - **Résultat:** `MNT_FRANCE_merged.vrt` (10 KB)
  - **À utiliser pour:** Commencer, c'est le plus important!
  - **Commande:** `python create_mnt_vrt.py`

- **`convert_vrt_to_geotiff.py`**
  - **Rôle:** Convertit le VRT en GeoTIFF compressé
  - **Résultat:** `MNT_FRANCE_merged.tif` (~150-200 MB)
  - **Prérequis:** GDAL installé (OSGeo4W ou Conda)
  - **Commande:** `python convert_vrt_to_geotiff.py`

- **`exemples_utilisation_mnt.py`**
  - **Rôle:** Montre comment utiliser le MNT en Python
  - **Contenu:** 4 exemples d'analyse géospatiale
  - **Commande:** `python exemples_utilisation_mnt.py`

#### 2️⃣ **Scripts Utilitaires**

- **`lanceur_fusion_mnt.bat`** (Windows)
  - Menu interactif pour lancer les scripts
  - Double-cliquez pour utiliser
  - Accès facile à toutes les options

- **`merge_mnt.bat`**
  - Script batch simple pour fusion directe
  - Alternative au script Python

#### 3️⃣ **Documentation**

- **`README.md`**
  - Documentation complète
  - Instructions détaillées
  - Guide d'utilisation
  - FAQ et dépannage

## 🚀 Démarrage Rapide (3 minutes)

### Étape 1: Créer le VRT (OBLIGATOIRE - 30 secondes)

```bash
python create_mnt_vrt.py
```

✅ Cela crée: `MNT_FRANCE_merged.vrt`

### Étape 2: Utiliser le VRT

**Option A - Visualiser dans QGIS (recommandé)**
1. Télécharger QGIS: https://qgis.org/download/
2. Ouvrir le fichier `MNT_FRANCE_merged.vrt`
3. Voir toutes les 291 tuiles comme un seul raster!

**Option B - Utiliser en Python**
```python
import rasterio
with rasterio.open(r'C:\Users\marya\Downloads\MNT_FRANCE_merged.vrt') as src:
    altitude = src.read(1)  # Lire les altitudes
    print(f"Min: {altitude.min()}, Max: {altitude.max()}")
```

### Étape 3 (Optionnel): Convertir en GeoTIFF

Si vous préférez un seul fichier raster (plus lourd):

```bash
python convert_vrt_to_geotiff.py
```

⚠️ Nécessite GDAL (voir section Installation plus bas)

## 📊 Comparaison des Formats

| Format | Taille | Avantages | Inconvénients |
|--------|--------|-----------|---------------|
| **ASC brut (291 fichiers)** | 2 GB | Natif IGN | Fragmenté |
| **VRT** | 10 KB | ✅ Léger, rapide, virtual | Dépend des ASC |
| **GeoTIFF** | 150-200 MB | ✅ Standalone, compressé | Plus lourd |

## ⚙️ Installation des Dépendances

### Dépendances Python Minimales (pour utiliser le VRT)

```bash
pip install rasterio numpy
```

C'est tout! Essayez et retournez à l'étape 1.

### Pour Convertir en GeoTIFF (optionnel)

Vous avez besoin de **GDAL**. Trois options:

#### Option 1: OSGeo4W (Windows - Recommandé)
1. Télécharger: https://trac.osgeo.org/osgeo4w/
2. Installer le paquet "GDAL"
3. Exécuter: `python convert_vrt_to_geotiff.py`

#### Option 2: Anaconda/Conda
```bash
conda install gdal
python convert_vrt_to_geotiff.py
```

#### Option 3: QGIS (Includes GDAL)
- QGIS contient GDAL
- Si vous avez QGIS, vous pouvez exporter le VRT en GeoTIFF directement!

## 🔍 Caractéristiques du MNT

| Paramètre | Valeur |
|-----------|--------|
| **Données** | MNT FRANCE - Modèle Numérique de Terrain |
| **Source** | IGN (Institut Géographique National) |
| **Format sources** | ESRI ASCII Grid (.asc) |
| **Nombre de tuiles** | 291 |
| **Résolution** | 5 mètres |
| **Couverture** | Région Département D001 (environ 75 km x 75 km) |
| **Projection** | Lambert93-IGN69 |
| **Altitude** | Format Float32 (altitudes en mètres) |
| **Valeur NoData** | -9999 |
