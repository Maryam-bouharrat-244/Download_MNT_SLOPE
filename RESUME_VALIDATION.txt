## 🎯 3 façons de valider

### 1️⃣ VALIDATION INTERACTIVE (UI web) - RECOMMANDÉ
**Parfait pour:** Débuter, tester des depts, validation progressive

```powershell
Double-cliquez: LANCER_APP.bat
OU
streamlit run mnt_app.py
```

**Interface web:**
```
Sélectionnez un département
        ↓
[🚀 Générer MNT VRT]
        ↓
Télécharge ↓ Extrait ↓ Génère VRT ↓ Valide
        ↓
Téléchargez le VRT
        ↓
Ouvrez dans QGIS
```

**Avantages:**
✅ Interface graphique belle  
✅ Barre de progression  
✅ Peut arrêter/reprendre  
✅ Validations visuelles  
✅ Export VRT inclus  

**Temps:** 15-25 min/département

---

### 2️⃣ VALIDATION BATCH (tous les depts) - PRODUCTION
**Parfait pour:** Production, tous les 96 depts, rapport complet

```powershell
Double-cliquez: VALIDER_TOUS_MNT.bat
OU
python validate_all_mnt.py
```

**Processus batch:**
```
Pour chaque département:
  1. Télécharge (8-15 min)
  2. Extrait (2-5 min)
  3. Cherche ASC (< 1 min)
  4. Génère VRT (1-3 min)
  5. Valide (< 1 min)
  6. Logs tout → CSV report
```

**Avantages:**
✅ Traite 96 depts unattended  
✅ Rapport CSV détaillé  
✅ Logs complets  
✅ Peut reprendre sur erreur  
✅ Statistiques complètes  

**Temps:** 48-72 heures (dépend connexion)

**Espace requis:** 100-200 GB

---

## ✅ Validations effectuées

### Téléchargement
```
✅ URL valide (IGN/GeoPF)
✅ Fichier 7z reçu
✅ Taille correcte (1-2 GB/dept)
✅ Hash/intégrité (optionnel)
```

### Extraction
```
✅ Archive 7z valide
✅ Structure RGEALTI trouvée
✅ Tous les dossiers en place
```

### Fichiers ASC
```
✅ Au moins 1 fichier trouvé (norm: 200-300)
✅ Noms format correct
✅ En-têtes lisibles
```

### Génération VRT
```
✅ Coordonnées Lambert93 valides
✅ Dimensions correctes
✅ NODATA lu dynamiquement
✅ Toutes les tuiles positionnées
✅ Fichier XML bien formé
```

### Validation VRT
```
✅ Format XML valide
✅ VRT structure correcte
✅ Projection EPSG:2154
✅ NoDataValue correct
✅ Sources(tuiles) présentes
```

## 🔗 Données sources

**Provider:** IGN (Institut National Géographique)  
**Format:** RGEALTI 2.0 - 5M ASC  
**Projection:** Lambert93 (EPSG:2154)  
**Résolution:** 5 mètres  
**Type:** Modèle Numérique de Terrain (Altitude)  
**URL base:** https://data.geopf.fr/telechargement/download/RGEALTI/
